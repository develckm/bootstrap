col 을 flex 정렬 하세요.

부트스트랩 Grid 시스템은 css flex 로 정의되어 있습니다. 때문에 row 는 display 가 flex 이고 col 은 flex item 입니다.

부트스트랩에서 디스플레이를 클래스에 d-block,d-inline,d-flex 등으로 쉽게 정의할 수 있는데 row는 이미 d-flex가 정의된 상태입니다.
